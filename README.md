# Análisis de Ventas – Alura Store

## Descripción
Este proyecto analiza los datos de ventas de cuatro tiendas con el objetivo de apoyar la toma de decisiones estratégicas.

## Objetivo
Determinar qué tienda presenta el menor desempeño general considerando ingresos, ventas por categoría, calificaciones de clientes, productos vendidos y costo de envío.

## Contenido del proyecto
- Análisis de ingresos por tienda
- Ventas por categoría
- Calificación promedio de clientes
- Productos más y menos vendidos
- Costo de envío promedio
- Análisis geográfico de ventas (extra)

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Cómo ejecutar
Abrir el notebook `.ipynb` en Google Colab y ejecutar las celdas en orden.

## Conclusión
Con base en los análisis realizados, se recomienda vender la Tienda 4, ya que presenta el menor impacto financiero dentro de la cadena.
